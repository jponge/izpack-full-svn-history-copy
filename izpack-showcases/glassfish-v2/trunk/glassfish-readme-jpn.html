GlassFish へようこそ！

このインストーラはシステムに GlassFish v2.1.1 b31g をセットアップします。

GlassFish は、Java EE 5 を実装するオープンソースのアプリケーションサーバです。Java EE 5 プラットフォームには、JavaServer Pages (JSP) 2.1、JavaServer Faces (JSF) 1.2、Servlet 2.5、Enterprise JavaBeans 3.0、Java API for Web Services (JAX-WS) 2.1、Java Architecture for XML Binding (JAXB) 2.1 など、数多くの最新技術が含まれます。

GlassFish についてさらに知りたい場合は、プロジェクトウェブサイト http://glassfish.org/ja/ をご覧下さい。

本配布について

本配布は Sun Microsystems による GlassFish サーバの正式なリリースではありません。正式な GlassFish バイナリパッケージの Julien Ponge による独自の再パッケージ化です。このインストーラはオープンソースの IzPack プロジェクト (http://izpack.org/) を使用し生成されています。
