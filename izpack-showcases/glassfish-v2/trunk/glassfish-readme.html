Welcome to GlassFish!

This will install and set-up GlassFish v2 ur2 b04 on your system.

GlassFish is a free, open source application server which implements the newest features in the Java EE 5 platform. The Java EE 5 platform includes the latest versions of technologies such as JavaServer Pages (JSP) 2.1, JavaServer Faces (JSF) 1.2, Servlet 2.5, Enterprise JavaBeans 3.0, Java API for Web Services (JAX-WS) 2.1, Java Architecture for XML Binding (JAXB) 2.1, and many other new technologies...

You can find out more about GlassFish on the project website at http://glassfish.org/.

About this distribution...

This is not an official release of the GlassFish application server by Sun Microsystems. This is an independent repackaging made by Julien Ponge from the official GlassFish binaries packaging. This installer was generated using the opensource IzPack project (see http://izpack.org/).
