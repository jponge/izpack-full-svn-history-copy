GlassFish へようこそ！

このインストーラはシステムに GlassFish v3 Prelude をインストールします。

本リリースは、Apache Felix OSGi ランタイム上に構築されていて、素晴らしい拡張性を提供します。Rails と Grails のサポート、開発者向け機能 (高速な起動、再配備時のセッション保持、NetBeans と Eclipse サポート、など)、組み込み API、Comet サポート、完全に新しい IPS ベースのアップデートセンター、RESTful ウェブサービス、いくつかの Java EE 6 機能のプレビュー、その他多くが含まれています。

GlassFish についてさらに知りたい場合は、プロジェクトウェブサイト http://glassfish.org/ja/ をご覧下さい。

本配布について

本配布は Sun Microsystems による GlassFish サーバの正式なリリースではありません。正式な GlassFish バイナリパッケージの Julien Ponge による独自の再パッケージ化です。このインストーラはオープンソースの IzPack プロジェクト (http://izpack.org/) を使用し生成されています。
