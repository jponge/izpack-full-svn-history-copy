Bienvenue dans GlassFish !

Ce logiciel va installer et configurer GlassFish v3 sur votre système.

GlassFish est un serveur d'application gratuit et open source qui implémente les toutes dernières nouveautés de Java EE 5. La plate-forme JavaEE 5 offre les dernières versions des technologies telles JavaServer Pages 2.1 (JSP), JavaServer Faces 1.2 (JSF), Servlets 2.5, Enterprise JavaBeans 3.0, Java API pour les Web Services (JAX-WS) 2.1, Java Architecture pour le binding XML 2.1 (JAXB) et plus encore ...

Pour de plus amples informations relatives au projet GlassFish, veuillez vous rendre sur le site du projet à l'addresse http://glassfish.org/.

A propos de cette distribution ...

Ceci n'est pas une distribution officielle du serveur d'application GlassFish par Sun Microsystems. Il s'agit d'une redistribution proposée par Julien Ponge à partir des distributions binaires officielles du projet GlassFish. Le présent installateur a été généré avec le logiciel open source IzPack disponible depuis http://izpack.org/.
