Welcome to GlassFish!

This will install and set-up GlassFish v3 Prelude on your system.

This release builds on top of the Apache Felix OSGi runtime, offers great extensibility, support for Rails and Grails, developer features (fast startup, retain sessions across redeploys, NetBeans and Eclipse tooling, ...), an embedded API, Comet support, a brand new IPS-based update center, RESTful web services, a preview of several Java EE 6 features, and much more.

You can find out more about GlassFish on the project website at http://glassfish.org/.

About this distribution...

This is not an official release of the GlassFish application server by Sun Microsystems. This is an independent repackaging made by Julien Ponge from the official GlassFish binaries packaging. This installer was generated using the opensource IzPack project (see http://izpack.org/).
