Welcome to Groovy!

This will install and setup Groovy 1.5.4 and some additional modules.

Groovy is an agile and dynamic language for the Java Virtual Machine. You can find out about Groovy at http://groovy.codehaus.org/.

About this Distribution

This is not an official distribution of Groovy. It is an independent repackaging from the official Groovy binary distribution, created using IzPack (see http://izpack.org/).
