1. What is JBrowser?
--------------------
JBrowser is a free 100% Java-based Web browser. It is intended to be integrated into an application, but can act as a standalone application.


2. Are there any system requirements?
-------------------------------------
 Java Runtime Environment 1.3 or higher installed.



3. Are there any known bugs or limitations?
-------------------------------------------
 Yes, JBrowser uses the javax.swing.JEditorPane class to view the HTML code. This component has many limitations, for example, it does not support Java applets, javascript, does not handle all html code correctly, etc.


5. I think I've found a bug, what should I do?
----------------------------------------------
 If you've found a bug, please get in touch with us. Please include this in your bug report:
1. The version of the software. Open the "About" window in the "Help" menu.
2. The version of the JRE. e.g. JDK 1.4 or J2SE ver 1.3.
3. Platform you are running on (e.g. Win 98 or Solaris 2.6).
4. Any exception stack trace, if one was generated.
5. A detailed description of how the problem occured.
